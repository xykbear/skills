# 输出格式模板

## 会话沉淀建议输出

```markdown
## 可添加到系统提示词的内容

### 1. [建议名称]
**来源**：从本次会话中识别

**内容**：
```xml
<TagName>
具体内容
</TagName>
```

**建议层级**：[根据上下文确定]
**理由**：为什么建议添加

---
添加到哪个位置？
- [ ] 外层配置（跨项目通用）
- [ ] 中间层配置（领域/工作区）
- [ ] 内层配置（项目特定）
- [ ] 其他（由用户指定）
```

## 审查报告输出

```markdown
## 审查报告：[文件名]

### 问题列表

| # | 问题 | 严重程度 | 建议 |
|---|------|----------|------|
| 1 | [问题描述] | 高/中/低 | [建议修改] |

### 优化建议

1. [建议1]
2. [建议2]

### 风险提示
[如涉及高风险修改，给出警示]
```
